HTML Calculator

We will code a simple calculator using HTML and Javascript in class. We will then add CSS code to style the calculator. The calculator will be styled using CSS with the following considerations:
• Functionality – how does it work?
• User experience – how will a user interact with it?
• User Familiarity – how do calculators usually look?
• Aesthetics – does the style add or detract from the function?
• The calculator will be styled in two ways: one freeform, and one focusing on the work of a designer.

The HTML Portrait

• Choose a portrait you took which focuses on the face.
• In Illustrator, create two drawings of the portrait using geometric shapes. Grayscale only.
• Change Illustrator preferences to work with pixels.
• Work with an artboard that is 1920px by 1080px.
• The drawing should be minimal and convey emotion (ex. Apple Animojis).
• Consider the types of shapes you can create in HTML – rectangles, circles, ovals and triangles.
• Once your drawing is approved, recreate it using only DIVs in HTML.
• Use DIVs position absolute, and start with the drawing aligning to the top and left of your browser.
